Detailed Description

This file uses interupt requests to quickly collect data from an RC circuit.

It uses two queues to collect time and ADC reading data. After collecting data This main file operates as a finite state machine: State 0 - sets Pin C1 high State 1 - prints ADC data, State 2 - sets Pin C1 low, State 3 - collects time data, State 4 - idle state.
